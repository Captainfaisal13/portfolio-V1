# Functionalities

- Search Your Favorite Movies and Tv Series
- Filter Movies and Tv Series by their Categories
- User Can Watch the Trailer of Particular Movie or Tv Series
- Realtime Essentail Information

# Features

- Get the Most Trending and Upcoming Movies
- Get the Direct Official Trailer
- Get the TMDB Ratings
- Get the Genres of Item
- Well Structured Design and User Experience
- Pagination

# Setup Details

If you want to run the project locally and want to contribute .

```console
# 1. Clone the repository
$ git clone https://github.com/Captainfaisal13/RokketBinge.git
# 2. Now head towards the root folder
$ cd ./CoinTracer
# 3. install the dependencies using npm or yarn
$ yarn install
or
$ npm install
# 4. Now run the project on your local machine
$ yarn dev
```
